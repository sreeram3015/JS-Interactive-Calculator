# JS-Interactive-Calculator

This project is an interactive calculator built using HTML, CSS, and JavaScript. It allows users to perform basic mathematical operations, such as addition, subtraction, multiplication, and division, on an interactive web interface.

## Features

- Addition, subtraction, multiplication, and division operations
- Interactive interface for a seamless user experience
- Clear screen functionality to reset the calculator

## Known Issues

- Subtraction operation is not working properly. Any guidance or assistance in resolving this issue is welcomed.

## Usage

To use the calculator, simply click the appropriate buttons on the interface to input numbers and operators. The output will be displayed in the designated display area.
